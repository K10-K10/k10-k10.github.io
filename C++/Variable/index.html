変数

変数とは？

そもそも変数とは、数値や文字を入れられる領域のことです。

変数の宣言

int a; <- これはint型のAという変数を宣言しています。

a = 3; <- これでAの中に3が入ります。

cout << a; <- これで3が出力されます。


変数の型

  • void

    • 型なし

  • int

    • 整数型(+-2biteまでの数値が入る)同じ整数型でlong longなどがある

  • string

    • 文字列

  • char

    • 1bite つまり 1文

  • bool

    • false,true つまり 0,1

  • struct

    • 構造体

  • union

    • 共用体

  • enum

    • 列挙型
