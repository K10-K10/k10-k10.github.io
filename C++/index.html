そもそもC++とは？

C++とは、もともとあるC言語を拡張した、オブジェクト指向プログラミング言語です。

C++の特徴としては、実行速度が速いことやメモリ管理が可能で、大規模開発に特化していることが挙げられます。

C++(コンパイラ)のインストール

Linuxでのインストール方法

sudo apt install g++

Windowsでのインストール方法

scoop install g++

C++の実行

C++の実行方法は、コマンドプロンプトで

g++ name.cpp # コンパイル
./a.out

または、コマンドプロンプトで

g++ name.cpp -o name # コンパイル
./name

C++はコンパイルという手順をふまないと実行出ません

または、VSCodeなどのエディターで実行できます。
