    • Pythonとは

    Pythohnは、知る限り一番使われているプログラミング言語です。

    読みやすい(?)、書きやすい(?)構文で、初心者の方でも始めやすいです。

    また、ライブラリが豊富で、機械学習など様々な分野で使われています。

    Pythonのインストール

    Linuxでのインストール方法

        sudo apt install python3

    Windowsでのインストール方法

            scoop install python

    Pythonの実行

    Pythonの実行方法は、コマンドプロンプトで

        python name.py

    または、VSCodeなどのエディターで実行できます。
