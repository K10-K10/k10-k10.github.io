VisualStudioCode解説

このページはVScodeの解説です。

コードエディタとは

        コードエディターとは、ソースコードを編集するためのソフトウェアのことです。
        VisualStudioCodeはコードエディタの一種です。
        ほかにはVisualStudioやAtom,vimなどもあります。

インストール方法

        VisualStudioCodeをインストールする方法は、
        公式ホームページこちらからダウンロードすることができます。

        
Linux

        また、aptを使ってインストールすることも可能です。
        aptとは、Linuxのソフトウェアをインストールするためのコマンドです。
        
            sudo apt install vscode
        


    
Windows

    windowsの場合はscoopを使ってインストールすることができます。
    
            scoop install vscode
        


    
macOS

    macOSの場合はbrewを使ってインストールすることができます。
    
            brew install vscode
