VisualStudioCode解説

このページはVScodeの解説です。

コードエディタとは

コードエディターとは、ソースコードを編集するためのソフトウェアのことです。 VisualStudioCodeはコードエディタの一種です。 ほかにはVisualStudioやAtom,vimなどもあります。

VisualStudioCode

VisualStudioCode(VSCode)は、コードエディタの一種で、Microsoftによって開発されています。 様々な拡張機能が特徴で、コンパイルや実行を便利にすることが可能になっています。 最近ではAIによる補完などもついかされ、コードを早く、簡単に書くことが可能になっています。

インストール方法

VisualStudioCodeをインストールする方法は、 公式ホームページこちらからダウンロードすることができます。

Linux

また、aptを使ってインストールすることも可能です。 aptとは、Linuxのソフトウェアをインストールするためのコマンドです。

            sudo apt install vscode

Windows

windowsの場合はscoopを使ってインストールすることができます。

        scoop install vscode

macOS

macOSの場合はbrewを使ってインストールすることができます。

        brew install vscode
